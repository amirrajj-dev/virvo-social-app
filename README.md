# 🌟 Virvo Social App 🌟

Welcome to **Virvo Social App**! A full-featured social media platform that allows users to share their thoughts, follow friends, and interact with posts. Virvo has a sleek and modern design with support for both dark and light modes.

## ✨ Features

- 🔒 **User authentication**: Sign Up, Login, Logout
- 📝 **Create, edit, and delete posts**
- 🤝 **Follow and unfollow users**
- 🔔 **Real-time notifications**
- 🌗 **Dark/Light mode toggle** using DaisyUI and TailwindCSS

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- Node.js
- npm
- Mongo Db
- mongo db compass (for better experience)

### 🔧 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/amirrajj-dev/virvo-social-app.git
cd virvo-social-app
npm run build
```

Start The Development Server:

```bash
npm run start
```

Open http://localhost:5000 to view it in the browser.
